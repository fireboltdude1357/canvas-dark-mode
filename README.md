# Canvas Dark Mode

A Chrome extension that applies a dark theme to Canvas LMS (Learning Management System).

## Features

- Dark theme applied instantly on page load (no flash of white)
- Toggle on/off via extension popup
- Preference saved and synced across devices
- Covers dashboard, courses, assignments, grades, quizzes, calendar, and more
- Dark scrollbars to match the theme

## Installation

### From source (Developer mode)

1. Clone or download this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top right)
4. Click **Load unpacked**
5. Select the `canvas-dark-mode` folder

## Usage

- The extension is **enabled by default**
- Click the extension icon in your toolbar to toggle dark mode on/off
- Changes apply immediately to open Canvas tabs

## Supported Sites

Works on all Canvas LMS instances hosted on `*.instructure.com`
